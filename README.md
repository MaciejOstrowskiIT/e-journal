# E-journal

E-journal project created for workshops / educational training organisation.

## Technologies Used

Project is created with:

- React
- React-Router
- REST API
- Express.js
- MongoDB
- Node.js

## Installation

```
$ git clone https://github.com/MaciejOstrowskiIT/e-journal.git
$ cd ../e-journal
$ npm install
$ npm run start

New terminal for middleware:

$ cd ./middleware
$ npm install
$ npm run dev

```
